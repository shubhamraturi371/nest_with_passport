
# Fundraiser

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository fundraiser project in nest app.

## Prerequisites

* node version v18.0.x above
* MySql v8.0.x

## Setup

* Clone this repo, checkout to the most active branch
```bash
git clone gitea@git.rubico.tech:Scott.Shubham/Node-fundraiser.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Environment handling

```bash
# copy .env.example file to .env 
$ cp .env.example .env

```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
